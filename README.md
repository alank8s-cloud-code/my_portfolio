## 🚀 Deployment Steps

### 1. Clone the Repository

```bash
git clone https://github.com/alank8s-cloud-code/my_portfolio.git
cd my_portfolio
```

### 2. Initialize & Push to GitHub

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/alank8s-cloud-code/my_portfolio.git
git push -u origin main
```

### 3. Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under **Source**, set:
   - Branch: `main`
   - Folder: `/root`
4. Click **Save**

### 4. Access Your Live Site

After a few seconds, your site will be live at:

```
https://alank8s-cloud-code.github.io/my_portfolio/
```
